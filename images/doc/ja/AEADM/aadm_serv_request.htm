Requesting a New Schema, More Storage, or Terminating Service

To make a make service request:

  1. Log in to Oracle Application Express. See "Logging In to Oracle Application Express" in Oracle Application Express Application Builder User's Guide.

    The Workspace home page appears.

  2. Click the Administration icon.

    The Administration page appears.

  3. Click Manage Service.

  4. Click Make a Service Request.

  5. Select one of the following:

    1. Request Schema

    2. Request Storage

    3. Request Termination

  6. If Request Schema, specify the following:

    1. Schema - Select either Request a new schema or Use an existing schema.

    2. Schema Name - Enter a new name or select one from the list.

    3. Click Next.

  7. If Request Storage, specify the following:

    1. Amount of Storage to Add - Select the amount of storage to add.

    2. Click Next.

    3. Confirm your request by clicking Request Storage.

  8. If Request Termination, specify the following:

    1. Verify the workspace service to be terminated.

    2. Click Next.

    3. Confirm your request by clicking Terminate Service.
