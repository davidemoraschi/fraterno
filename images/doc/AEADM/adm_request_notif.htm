Viewing Requests on the Instance Administration Home Page

Instance administrators can view pending and new service requests on the Oracle Application Express Instance Administration (Instance Administration) home page).

The Pending Requests region lists the total number of new service requests and service change requests. To view a more detailed report, click the number listed.


Description of the illustration adm_pend_requests.gif

The New Service Requests report displays at the bottom on the page. This report lists service requests by date. To view a specific request, click the description.
