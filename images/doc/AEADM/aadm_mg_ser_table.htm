Managing Websheet Database Objects

Use the Manage Export Repository page to manage files in the current workspace. All Websheet application data is stored within tables owned by a schema associated with this workspace. The tables are all prefixed by APEX$_. Use this page to manage these tables.

To access the Manage Export Repository page:

  1. Navigate to the Workspace home page.

  2. Click the Administration icon.

    The Workspace Administration page appears.

  3. On the Tasks list, click Websheet Database Objects.

    The Websheet Database Objects page appears.

  4. Select one of the following:

    • Create Websheet Database Objects - Available only if objects not created

    • Validate Websheet Database Objects
